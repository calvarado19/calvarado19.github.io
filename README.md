# Modern Personal Portfolio

React + Vite + TailwindCSS portfolio for Carlos Alvarado. Features sections for hero, highlights, projects, skills, certifications, and contact with quick social CTAs.

## Local setup

```bash
# install dependencies
HTTP_PROXY= HTTPS_PROXY= npm install

# start dev server
HTTP_PROXY= HTTPS_PROXY= npm run dev

# run lint
HTTP_PROXY= HTTPS_PROXY= npm run lint

# build for production
HTTP_PROXY= HTTPS_PROXY= npm run build
```

The `HTTP_PROXY= HTTPS_PROXY=` prefix clears inherited proxy environment variables in this workspace. Remove it if you already have working network access.
