# calvarado19.github.io

## Local development

This site uses Jekyll (GitHub Pages-compatible). From the repository root:

1. Install dependencies:
   ```bash
   bundle install
   ```
2. Build the site:
   ```bash
   bundle exec jekyll build
   ```
3. Serve locally:
   ```bash
   bundle exec jekyll serve --livereload
   ```

If your environment blocks gem downloads, run these commands on a machine with normal internet access.
